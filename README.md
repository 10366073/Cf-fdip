要查看Linux系统的架构，你可以使用以下命令之一：

使用 uname 命令：
uname -m
这将显示机器的硬件架构信息，例如 x86_64 表示 64 位架构，i386 表示 32 位架构。

然后去下载对应的版本。（我这里的是arm64）
